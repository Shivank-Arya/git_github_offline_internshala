**Project Structure

src/
├── index.html          (Home page with link to blog)
├── blog.html           (Blog page with article cards)
├── blog.css            (Styling for blog cards & layout)
├── article.css         (Styling for individual article pages)
└── blog/
    ├── article1.html   (Getting Started with Web Development)

Key Features
✨ Blog Main Page (blog.html)

Beautiful card layout with 6 sample articles
Article previews with metadata (date, category, read time)
Colorful gradient backgrounds for each card
Hover effects with smooth animations
"Read More" buttons linking to full articles
📄 Individual Article Pages

Full article content with proper formatting
Navigation back to blog homepage
Consistent styling and responsive design
Metadata display (date, category, read time)
🎨 Responsive Design

Works on mobile, tablet, and desktop
Responsive grid layout (auto-adapts from 1-3 columns)
Mobile-friendly navigation
Optimized typography for readability
🔗 Navigation

Sticky navbar with logo and links
Easy linking between Home → Blog → Articles
All pages are interconnected
How to Use
Open index.html to see the home page
Click "Explore My Blog" to go to blog.html
Click "Read More →" on any card to read the full article
Use the back button to return to the blog list
You can easily customize this by:

Editing article content in the blog/ folder
Adding more cards to blog.html by copying the card structure
Changing colors and gradients in the CSS files
Replacing the gradient backgrounds with actual images
